# Finance Dashboard

A simple and interactive finance dashboard to track income, expenses, and overall financial activity in one place.

This project was built to demonstrate frontend skills like UI design, data handling, and chart visualization using JavaScript.

## What this project does

This dashboard allows users to:
- View total balance, income, and expenses  
- Analyze spending using charts  
- Manage and organize transactions  
- Get quick financial insights  

## Features

- Dashboard with real-time calculations  
- Charts (Income vs Expense, Category breakdown)  
- Add transactions  
- Edit and delete transactions  
- Search functionality  
- Sort by date and amount  
- Role-based access (Admin / Viewer)  
- Dark mode toggle  
- Export data as CSV  

## Built With

- HTML  
- CSS (custom styles in style.css)  
- Bootstrap  
- JavaScript (script.js)  
- Chart.js

## How to run

1. Download or clone the repository  
2. Open the project folder  
3. Open index.html in your browser  

No installation required.

## Notes

- Admin role allows adding, editing, and deleting transactions  
- Viewer role is read-only  
- Data is currently stored in JavaScript (no backend)
